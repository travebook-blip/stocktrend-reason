# 更新紀錄 CHANGELOG

版本號顯示在網站標題右側（游標移上去看更新內容），頁尾也有同樣資訊。
每次要更新 `index.html`，先看這裡確認你現在的版本、以及新版改了什麼。

---

## v1.3.0 — 2026-07-01
**控盤研判改為同時顯示近20日／近60日雙區間**
- 每張卡片現在顯示兩組控盤天平：近20日（短線）與近60日（波段），不再只給單一結論
- 兩區間結論不同時，跳出⚠提示，建議以較長區間（60日）為主要參考
- 法人買賣超表格新增「近60日」欄位
- 上方摘要統計與篩選鈕（外資控盤／投信控盤）改以60日為準（資料不足60日時退回20日）
- 網站標題新增版本號顯示，方便確認是否為最新版

## v1.2.0 — 2026-07-01
**簡化清單管理，移除雲端同步（Cloudflare Worker）**
- 拿掉「雲端清單同步」面板與相關程式碼（GitHub token、Worker 串接）
- 保留手動輸入 / CSV 匯入 / 貼上範例 / 複製清單 / 清除，維持零設定即可使用
- 要更新每日自動排程的清單，改為：網站「複製清單」→ 貼到 GitHub 網頁編輯 `watchlist.txt`

## v1.1.0 — 2026-06-30
**清單管理功能**
- 新增手動輸入清單提示文字
- 新增「清除」按鈕
- 新增「複製清單」按鈕（輸出成 watchlist.txt 格式）
- 清單自動記憶於瀏覽器（localStorage）

## v1.0.0 — 2026-06-29
**首次發布**
